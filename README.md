# Installation

Developed and tested using Python 3.8.6

## Create a virtual environment

```
python -m venv venv
```

## Enable in (MacOS)

```
source venv/bin/activate
```

## Install dependencies

```
pip install -r requirements.txt
```

# Run it

## Start server

```
python server.py
```

## Start multiple clients in one go

```
python multiprocess.py
```

## Start client

```
python client.py
```
